## 1. Block Overview
The APB_UART is a Universal Asynchronous Receiver/Transmitter (UART) block with an APB slave interface for register access. It provides a serial communication channel in SoC designs, supporting full-duplex UART communication, programmable baud rate, configurable data frame format, and hardware flow control. The block is targeted at applications requiring low-speed serial data communication with external peripherals such as GPS modules, Bluetooth chips, and debug consoles. It is clocked from the APB bus clock (pclk) and operates as a single clock domain design.

## 11. Known Constraints
- Maximum supported baud rate is 115200 bps when pclk = 50 MHz
- TX and RX FIFOs share the same depth parameter (FIFO_DEPTH); they cannot be independently sized
- No DMA interface supported in this version (v1.0)
- pslverr is always driven low — no error response supported in v1.0
