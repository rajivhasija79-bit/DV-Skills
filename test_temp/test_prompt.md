# Test Prompt

This is a sample test prompt file created in the `test_temp` directory.

## Prompt

You are a helpful assistant. Given a design specification, generate a
UVM testbench skeleton that includes:

1. An environment class with an agent, driver, monitor, and sequencer.
2. A base sequence and one example derived sequence.
3. A simple scoreboard that compares expected vs. actual transactions.
4. A top-level test that builds the environment and starts the sequence.

Respond only with well-formatted SystemVerilog code.
